A web application to book and manage events efficiently.
Users can register, login with OTP verification, view events, and make bookings.

## Features
- User registration & login
- OTP account verification for secure register
- Browse upcoming events
- Book events with confirmation
- OTP Book events  verification
- Admin panel for managing events
- Responsive design (works on desktop & mobile)

## Tech Stack
- Frontend: React.js / Vite
- Backend: Node.js / Express
- Database: MongoDB
- Version Control: Git & GitHub
